Tree

Linux utility program

Source Code

The tree is a tiny, cross-platform command-line program used to recursively list or display the content of a directory in a tree-like format. It outputs the directory paths and files in each sub-directory and a summary of a total number of sub-directories and files. There is also option to sort the output using time or name and to store output in JSON format.
